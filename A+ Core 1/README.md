# 🔧 CompTIA A+ Core 1 (220-1101)

This section of the repository documents my study progress for the CompTIA A+ Core 1 certification.

## 📘 Overview

Core 1 focuses on the physical and connectivity aspects of IT, including hardware, networking, mobile devices, cloud computing, and troubleshooting.

## 🎯 Objectives

* Understand core hardware components and their functions
* Learn networking fundamentals (ports, protocols, IP addressing)
* Gain familiarity with mobile devices and their components
* Explore virtualization and cloud computing concepts
* Develop structured troubleshooting skills

## 🛠 Structure

Each module includes:

* **Key Concepts** → concise notes of important topics
* **Lab** → hands-on practice or recreated scenario
* **Troubleshooting Scenario** → real-world problem simulation
* **Reflection** → lessons learned and improvements

## 🚀 Goals

* Build a strong foundation for IT support roles
* Prepare for the CompTIA A+ Core 1 exam (220-1101)
* Develop practical troubleshooting skills through hands-on labs
