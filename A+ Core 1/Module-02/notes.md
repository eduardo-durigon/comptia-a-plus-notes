# 2.1.1 Personal Computers

### PC Components

Personal computers are divided into:

* External peripherals
* Internal system components

### Internal Components Include

* Motherboard
* CPU
* RAM
* Storage drives
* Power supply unit (PSU)

### PC Case Features

A standard tower case includes:

* Front I/O panel
* Audio and USB ports
* Power/reset buttons
* Drive bays
* Cooling fans and vents

### Rear Panel Components

* PSU socket
* Motherboard I/O shield
* Expansion card slots
* Cooling exhaust fans

### Important Concepts

Proper case assembly helps prevent:

* Dust buildup
* Overheating
* Electrostatic discharge (ESD)
* Electromagnetic interference (EMI)

---

# 2.1.2 Peripheral Devices and Ports

### Peripheral Devices

Peripherals connect to the PC through I/O ports.

### Common Ports

* USB
* HDMI
* DisplayPort
* Audio ports
* RJ-45 Ethernet
* SATA

### Key Concepts

* Ports are external connection points
* Connectors fit into matching ports
* Most modern connectors are keyed or reversible to prevent incorrect insertion

---

# 2.1.3 Binary Data and Transfer Units

### Binary Storage

Computers process binary data using:

* Bits (b)
* Bytes (B)

### Data Units

| Unit             | Measurement   |
| ---------------- | ------------- |
| Kilobit/Kilobyte | 1,000         |
| Megabit/Megabyte | 1,000,000     |
| Gigabit/Gigabyte | 1,000,000,000 |

### Important Reminder

* Lowercase **b** = bits
* Uppercase **B** = bytes

---

# 2.1.4 Universal Serial Bus (USB)

USB is the standard interface for connecting peripherals.

### USB Connector Types

* **Type-A** → Standard rectangular connector
* **Type-B** → Printer-style connector
* **Mini-USB** → Older small devices
* **Micro-USB** → Smartphones/tablets
* **USB-C** → Modern reversible connector

### USB-C Advantages

* Reversible design
* Faster transfer speeds
* Power delivery support
* Supports video output

### USB Power Delivery

USB can provide power to connected devices:

* Standard USB → up to 4.5W
* USB Power Delivery (PD) → up to 100W

---

# 2.1.6 Display Types

### LCD Panel Types

#### IPS (In-Plane Switching)

**Pros**

* Excellent color accuracy
* Wide viewing angles

**Cons**

* Slower response times

---

#### TN (Twisted Nematic)

**Pros**

* Fast refresh rates
* Good for gaming

**Cons**

* Poor color accuracy
* Limited viewing angles

---

#### VA (Vertical Alignment)

**Pros**

* Better contrast than TN
* Good color reproduction

**Cons**

* Possible color shifting

---

### OLED Displays

#### Advantages

* True blacks
* Excellent contrast
* Better color accuracy
* Energy efficient

#### Disadvantages

* Risk of burn-in

---

### Mini-LED Displays

#### Advantages

* Improved brightness
* Better contrast
* Reduced halo effect

#### Disadvantages

* More expensive
* Does not achieve true blacks like OLED

---

# 2.1.8 HDMI and DisplayPort

# 2.1.9 Thunderbolt Interface

Thunderbolt combines:

* Data transfer
* Video output
* Peripheral connectivity

### Versions

| Version         | Features                            |
| --------------- | ----------------------------------- |
| Thunderbolt 1/2 | Mini DisplayPort connector          |
| Thunderbolt 3   | USB-C connector, up to 40 Gbps      |
| Thunderbolt 4   | Improved minimum performance        |
| Thunderbolt 5   | Higher speeds and expanded features |

### Important Notes

* Not all USB-C ports support Thunderbolt
* Look for the lightning bolt icon

---

# 2.1.10 Lightning Interface

### Overview

Apple’s Lightning connector was introduced in 2012.

### Features

* Compact reversible connector
* Used on older iPhones and iPads
* Supports charging and data transfer

### Industry Shift

Lightning is gradually being replaced by USB-C because:

* Faster speeds
* Better power delivery
* Industry standardization

---

### SATA Speeds

| SATA Revision | Speed    |
| ------------- | -------- |

| SATA II       | 300 MBps |
| SATA III      | 600 MBps |

* 4-pin Molex connectors

# 2.1.13 eSATA

### Features

* External SATA connection
* Supports external drives
* Longer cable lengths than internal SATA

# 2.2 Motherboards

## 2.2.1 Motherboard Functions

### Main Purpose

The motherboard connects all computer components.

### Key Components

* CPU
* RAM
* Storage
* Expansion cards
* Chipset
* Bus interfaces

### Important Concepts

* System clock synchronizes operations
* Data travels across buses
* CPU compatibility depends on socket/chipset

---

# 2.2.2 Electrical Safety and ESD

### Safety Procedures

Before working inside a PC:

* Unplug power
* Drain residual power
* Use anti-static protection

### ESD Prevention

* Use anti-ESD wrist straps
* Handle components by edges
* Avoid touching contacts or chips

---

# 2.2.3 CPU and Memory Connectors

### CPU Socket

Different CPUs require different socket types:

* Intel sockets
* AMD sockets

### RAM Slots

Modern RAM types include:

* DDR3
* DDR4
* DDR5

### Important Notes

* DIMMs fit into motherboard slots
* RAM slots are color-coded and keyed

---

# 2.2.4 Storage Interfaces

### Storage Types

* HDD
* SSD
* NVMe

### M.2 Interface

M.2 SSDs:

* Mount directly to motherboard
* Require no data cable
* Provide faster performance

### eSATA

Provides external SATA connectivity but is less common today.

---

# 2.2.5 PCI Express (PCIe)

PCIe is the modern expansion interface for:

* GPUs
* Sound cards
* Network cards
* Capture cards

### PCIe Slot Types

* x1
* x4
* x8
* x16

### Important Concepts

* More lanes = more bandwidth
* PCIe versions are backward compatible
* Shared bandwidth can reduce slot speeds

---

# 2.2.6 Motherboard Form Factors

### ATX

* Standard desktop motherboard
* Up to 7 expansion slots

### Micro-ATX

* Smaller version of ATX
* Up to 4 expansion slots

### Mini-ITX

* Compact form factor
* Usually 1 expansion slot

---

# 2.2.7 Motherboard Installation

### Installation Steps

1. Review documentation
2. Install I/O shield
3. Install standoffs
4. Install CPU and RAM
5. Secure motherboard
6. Connect power/data cables
7. Manage cables for airflow

### Important Reminder

Incorrect standoff placement can short-circuit the motherboard.

# 2.3 Legacy Cables

## DVI and VGA

### DVI

Supports:

* Digital signals
* Analog signals (depending on type)

### DVI Types

* DVI-A
* DVI-D
* DVI-I

---

### VGA

### Features

* Analog-only video connection
* Older CRT monitor standard
* Uses 15-pin connector

### Limitations

* Lower image quality than digital standards

---

# 2.3.1 Serial Cables

### RS-232 Serial Ports

Used for:

* Older modems
* Device management
* Legacy networking equipment

### Common Connector

* DB9 connector

---

# 2.3.2 Adapter Cables

### Purpose

Adapter cables connect incompatible ports or standards.

### Common Examples

* HDMI to VGA
* HDMI to DVI
* USB-C to USB-A
* USB-C to HDMI

### Important Difference

* Active adapters convert signals
* Passive adapters only change connector type

---

# Key Takeaways

* USB-C and Thunderbolt are modern high-speed standards replacing many legacy connectors.
* Display technologies differ greatly in color accuracy, refresh rates, and contrast.
* Motherboards determine CPU compatibility, RAM support, and expansion capabilities.
* PCIe is the standard interface for modern expansion cards.
* Proper ESD protection is essential when handling computer hardware.
* SATA, M.2, and NVMe are important storage connection technologies.
* Legacy interfaces like VGA and serial ports still appear in some environments but are becoming obsolete.
